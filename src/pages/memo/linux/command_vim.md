In console :
```bash
vim [OPTIONS] filename
```

Interactive mode :
| Action | Key | Example | Comments |
|---|---|---|---|
| Quit | `:q` | - | - |
| Insert | `:i` | - | - |
| Save | `:w` | - | - |
| Force Save | `:w!` | - | - |
